# classmates

### 采用vue-cli脚手架创建文件

### front-end: vue + vue-router + vuex + vue-resource

### ui库：mint-ui + (部分二次开发)

#### 文件说明：

#### assets/sass  scss文件

#### pages        页面文件

#### components   共用组件

#### uicomponents 二次开发的ui组件

#### store        vuex文件

#### config.js  一般的配置文件

#### utils.js  共用接口文件


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
